# MCQ App (React + Vite)

A simple multiple-choice quiz app built with React.

## Features

- 4 sample MCQ questions
- One answer selection per question
- Score tracking
- Final result with percentage
- Restart quiz button

## Run Locally

1. Install dependencies:

   npm install

2. Start the development server:

   npm run dev

3. Open the local URL shown in your terminal (usually http://localhost:5173).

## Build

To create a production build:

npm run build
